<p>In the <a href="/wiki/Mathematics" title="Mathematics">mathematical</a> field of <a href="/wiki/Dynamical_system" title="Dynamical system">dynamical systems</a>, an <b>attractor</b> is a set of numerical values toward which a system tends to evolve, for a wide variety of starting conditions of the system. System values that get close enough to the attractor values remain close even if slightly disturbed.</p>
<p>In finite-dimensional systems, the evolving variable may be represented <a href="/wiki/Algebra" title="Algebra">algebraically</a> as an <i>n</i>-dimensional <a href="/wiki/Coordinate_vector" title="Coordinate vector">vector</a>. The attractor is a region in <a href="/wiki/Space_(mathematics)" title="Space (mathematics)"><i>n</i>-dimensional space</a>. In <a href="/wiki/Physics" title="Physics">physical systems</a>, the <i>n</i> dimensions may be, for example, two or three positional coordinates for each of one or more physical entities; in <a href="/wiki/Economics" title="Economics">economic systems</a>, they may be separate variables such as the <a class="mw-redirect" href="/wiki/Inflation_rate" title="Inflation rate">inflation rate</a> and the <a class="mw-redirect" href="/wiki/Unemployment_rate" title="Unemployment rate">unemployment rate</a>.</p>
<p>If the evolving variable is two- or three-dimensional, the attractor of the dynamic process can be represented <a href="/wiki/Geometry" title="Geometry">geometrically</a> in two or three dimensions, (as for example in the three-dimensional case depicted to the right). An attractor can be a <a href="/wiki/Point_(geometry)" title="Point (geometry)">point</a>, a finite set of points, a <a href="/wiki/Curve" title="Curve">curve</a>, a <a href="/wiki/Manifold" title="Manifold">manifold</a>, or even a complicated set with a <a href="/wiki/Fractal" title="Fractal">fractal</a> structure known as a <i><a href="/wiki/Attractor#Strange_attractor" title="Attractor">strange attractor</a>.</i> If the variable is a <a href="/wiki/Scalar_(mathematics)" title="Scalar (mathematics)">scalar</a>, the attractor is a subset of the real number line. Describing the attractors of chaotic dynamical systems has been one of the achievements of <a href="/wiki/Chaos_theory" title="Chaos theory">chaos theory</a>.</p>
<p>A <a href="/wiki/Trajectory" title="Trajectory">trajectory</a> of the dynamical system in the attractor does not have to satisfy any special constraints except for remaining on the attractor, forward in time. The trajectory may be <a href="/wiki/Periodic_function" title="Periodic function">periodic</a> or <a href="/wiki/Chaos_theory" title="Chaos theory">chaotic</a>. If a set of points is periodic or chaotic, but the flow in the neighborhood is away from the set, the set is not an attractor, but instead is called a <b>repeller</b> (or <i><a class="mw-redirect" href="/wiki/Repellor" title="Repellor">repellor</a></i>).</p>

<h2><span class="mw-headline" id="Motivation">Motivation</span></h2>
<p>A <a href="/wiki/Dynamical_system" title="Dynamical system">dynamical system</a> is generally described by one or more <a class="mw-redirect" href="/wiki/Differential_equations" title="Differential equations">differential</a> or <a class="mw-redirect" href="/wiki/Difference_equations" title="Difference equations">difference equations</a>. The equations of a given dynamical system specify its behavior over any given short period of time. To determine the system's behavior for a longer period, it is often necessary to <a href="/wiki/Integral" title="Integral">integrate</a> the equations, either through analytical means or through iteration, often with the aid of computers.</p>
<p>Dynamical systems in the physical world tend to arise from <a href="/wiki/Dissipative_system" title="Dissipative system">dissipative</a> systems: if it were not for some driving force, the motion would cease. (Dissipation may come from <a href="/wiki/Friction" title="Friction">internal friction</a>, <a href="/wiki/Thermodynamics" title="Thermodynamics">thermodynamic losses</a>, or loss of material, among many causes.) The dissipation and the driving force tend to balance, killing off initial transients and settle the system into its typical behavior. The subset of the <a href="/wiki/Phase_space" title="Phase space">phase space</a> of the dynamical system corresponding to the typical behavior is the <b>attractor</b>, also known as the attracting section or attractee.</p>
<p>Invariant sets and <a href="/wiki/Limit_set" title="Limit set">limit sets</a> are similar to the attractor concept. An <i>invariant set</i> is a set that evolves to itself under the dynamics. Attractors may contain invariant sets. A <i>limit set</i> is a set of points such that there exists some initial state that ends up arbitrarily close to the limit set (i.e. to each point of the set) as time goes to infinity. Attractors are limit sets, but not all limit sets are attractors: It is possible to have some points of a system converge to a limit set, but different points when perturbed slightly off the limit set may get knocked off and never return to the vicinity of the limit set.</p>
<p>For example, the <a href="/wiki/Pendulum#damped_pendulum" title="Pendulum">damped pendulum</a> has two invariant points: the point <span class="texhtml">x<sub>0</sub></span> of minimum height and the point <span class="texhtml">x<sub>1</sub></span> of maximum height. The point <span class="texhtml">x<sub>0</sub></span> is also a limit set, as trajectories converge to it; the point <span class="texhtml">x<sub>1</sub></span> is not a limit set. Because of the dissipation due to air resistance, the point <span class="texhtml">x<sub>0</sub></span> is also an attractor. If there was no dissipation, <span class="texhtml">x<sub>0</sub></span> would not be an attractor.</p>

<h2><span class="mw-headline" id="Types_of_attractors">Types of attractors</span></h2>
<p>Attractors are portions or <a href="/wiki/Subset" title="Subset">subsets</a> of the <a href="/wiki/Configuration_space#Phase_space" title="Configuration space">phase space</a> of a <a href="/wiki/Dynamical_system" title="Dynamical system">dynamic system</a>. Until the 1960s, attractors were thought of as being <a href="/wiki/Geometric_primitive" title="Geometric primitive">simple geometric subsets</a> of the phase space, like <a href="/wiki/Point_(geometry)" title="Point (geometry)">points</a>, <a class="mw-redirect" href="/wiki/Line_(mathematics)" title="Line (mathematics)">lines</a>, <a href="/wiki/Surface_(topology)" title="Surface (topology)">surfaces</a>, and simple regions of <a href="/wiki/Three-dimensional_space" title="Three-dimensional space">three-dimensional space</a>. More complex attractors that cannot be categorized as simple geometric subsets, such as <a href="/wiki/Topology" title="Topology">topologically</a> wild sets, were known of at the time but were thought to be fragile anomalies. <a href="/wiki/Stephen_Smale" title="Stephen Smale">Stephen Smale</a> was able to show that his <a href="/wiki/Horseshoe_map" title="Horseshoe map">horseshoe map</a> was <a href="/wiki/Structural_stability" title="Structural stability">robust</a> and that its attractor had the structure of a <a href="/wiki/Cantor_set" title="Cantor set">Cantor set</a>.</p>
<p>Two simple attractors are a <a href="/wiki/Fixed_point_(mathematics)" title="Fixed point (mathematics)">fixed point</a> and the <a href="/wiki/Limit_cycle" title="Limit cycle">limit cycle</a>. Attractors can take on many other geometric shapes (phase space subsets). But when these sets (or the motions within them) cannot be easily described as simple combinations (e.g. <a href="/wiki/Intersection_(set_theory)" title="Intersection (set theory)">intersection</a> and <a href="/wiki/Union_(set_theory)" title="Union (set theory)">union</a>) of <a href="/wiki/Geometric_primitive" title="Geometric primitive">fundamental geometric objects</a> (e.g. <a class="mw-redirect" href="/wiki/Line_(mathematics)" title="Line (mathematics)">lines</a>, <a href="/wiki/Surface_(topology)" title="Surface (topology)">surfaces</a>, <a href="/wiki/Sphere" title="Sphere">spheres</a>, <a href="/wiki/Toroid" title="Toroid">toroids</a>, <a href="/wiki/Manifold" title="Manifold">manifolds</a>), then the attractor is called a <i><a href="/wiki/Attractor#Strange_attractor" title="Attractor">strange attractor</a></i>.</p>
<h3><span class="mw-headline" id="Fixed_point">Fixed point</span></h3>

<p>A <a href="/wiki/Fixed_point_(mathematics)" title="Fixed point (mathematics)">fixed point</a> of a function or transformation is a point that is mapped to itself by the function or transformation. If we regard the evolution of a dynamical system as a series of transformations, then there may or may not be a point which remains fixed under each transformation. The final state that a dynamical system evolves towards corresponds to an attracting fixed point of the evolution function for that system, such as the center bottom position of a <a href="/wiki/Pendulum#damped_pendulum" title="Pendulum">damped pendulum</a>, the level and flat water line of sloshing water in a glass, or the bottom center of a bowl contain a rolling marble. But the fixed point(s) of a dynamic system is not necessarily an attractor of the system. For example, if the bowl containing a rolling marble was inverted and the marble was balanced on top of the bowl, the center bottom (now top) of the bowl is a fixed state, but not an attractor. This is equivalent to the difference between <a href="/wiki/Stability_theory#Stability_of_fixed_points" title="Stability theory">stable and unstable equilibria</a>. In the case of a marble on top of an inverted bowl (a hill), that point at the top of the bowl (hill) is a fixed point (equilibrium), but not an attractor (stable equilibrium).</p>
<p>In addition, physical dynamic systems with at least one fixed point invariably have multiple fixed points and attractors due to the reality of dynamics in the physical world, including the <a class="mw-redirect" href="/wiki/Nonlinear_dynamics" title="Nonlinear dynamics">nonlinear dynamics</a> of <a href="/wiki/Stiction" title="Stiction">stiction</a>, <a href="/wiki/Friction" title="Friction">friction</a>, <a href="/wiki/Surface_roughness" title="Surface roughness">surface roughness</a>, <a href="/wiki/Deformation_(engineering)" title="Deformation (engineering)">deformation</a> (both <a class="mw-redirect" href="/wiki/Elastic_deformation" title="Elastic deformation">elastic</a> and <a href="/wiki/Plastic" title="Plastic">plasticity</a>), and even <a href="/wiki/Quantum_mechanics" title="Quantum mechanics">quantum mechanics</a>. In the case of a marble on top of an inverted bowl, even if the bowl seems perfectly <a href="/wiki/Sphere#Hemisphere" title="Sphere">hemispherical</a>, and the marble's <a href="/wiki/Sphere" title="Sphere">spherical</a> shape, are both much more complex surfaces when examined under a microscope, and their <a href="/wiki/Contact_mechanics#History" title="Contact mechanics">shapes change</a> or <a href="/wiki/Deformation_(mechanics)" title="Deformation (mechanics)">deform</a> during contact. Any physical surface can be seen to have a rough terrain of multiple peaks, valleys, saddle points, ridges, ravines, and plains. There are many points in this surface terrain (and the dynamic system of a similarly rough marble rolling around on this microscopic terrain) that are considered <a href="/wiki/Critical_point_(mathematics)" title="Critical point (mathematics)">stationary</a> or fixed points, some of which are categorized as attractors.</p>
<h3><span class="mw-headline" id="Finite_number_of_points">Finite number of points</span></h3>
<p>In a discrete-time system, an attractor can take the form of a finite number of points that are visited in sequence. Each of these points is called a <a href="/wiki/Periodic_point" title="Periodic point">periodic point</a>. This is illustrated by the <a href="/wiki/Logistic_map" title="Logistic map">logistic map</a>, which depending on its specific parameter value can have an attractor consisting of 2 points, 3×2 points, etc., for any value of <i>n</i>.</p>
<h3><span class="mw-headline" id="Limit_cycle">Limit cycle</span></h3>

<p>A <a href="/wiki/Limit_cycle" title="Limit cycle">limit cycle</a> is a periodic orbit of a continuous dynamical system that is <a href="/wiki/Isolated_point" title="Isolated point">isolated</a>. Examples include the swings of a <a href="/wiki/Pendulum_clock" title="Pendulum clock">pendulum clock</a>, the tuning circuit of a radio, and the heartbeat while resting. (The limit cycle of an ideal pendulum is not an example of a limit cycle attractor because its orbits are not isolated: in the phase space of the ideal pendulum, near any point of a periodic orbit there is another point that belongs to a different periodic orbit, so the former orbit is not attracting).</p>

<h3><span class="mw-headline" id="Limit_torus">Limit torus</span></h3>
<p>There may be more than one frequency in the periodic trajectory of the system through the state of a limit cycle. For example, in physics, one frequency may dictate the rate at which a planet orbits a star while a second frequency describes the oscillations in the distance between the two bodies. If two of these frequencies form an <a href="/wiki/Irrational_number" title="Irrational number">irrational fraction</a> (i.e. they are <a href="/wiki/Commensurability_(mathematics)" title="Commensurability (mathematics)">incommensurate</a>), the trajectory is no longer closed, and the limit cycle becomes a limit <a href="/wiki/Torus" title="Torus">torus</a>. This kind of attractor is called an <span class="texhtml"><i>N</i><sub><i>t</i></sub></span>-torus if there are <span class="texhtml">N<sub>t</sub></span> incommensurate frequencies. For example, here is a 2-torus:</p>

<p>A time series corresponding to this attractor is a <a class="mw-redirect" href="/wiki/Quasiperiodic" title="Quasiperiodic">quasiperiodic</a> series: A discretely sampled sum of <span class="texhtml">N<sub>t</sub></span> periodic functions (not necessarily <a href="/wiki/Sine" title="Sine">sine</a> waves) with incommensurate frequencies. Such a time series does not have a strict periodicity, but its <a class="mw-redirect" href="/wiki/Power_spectrum" title="Power spectrum">power spectrum</a> still consists only of sharp lines.</p>
<h3><span class="mw-headline" id="Strange_attractor">Strange attractor</span></h3>

<p>An attractor is called <b>strange</b> if it has a <a href="/wiki/Fractal" title="Fractal">fractal</a> structure. This is often the case when the dynamics on it are <a href="/wiki/Chaos_theory" title="Chaos theory">chaotic</a>, but <a href="/wiki/Strange_nonchaotic_attractor" title="Strange nonchaotic attractor">strange nonchaotic attractors</a> also exist. If a strange attractor is chaotic, exhibiting <a class="mw-redirect" href="/wiki/Sensitive_dependence_on_initial_conditions" title="Sensitive dependence on initial conditions">sensitive dependence on initial conditions</a>, then any two arbitrarily close alternative initial points on the attractor, after any of various numbers of iterations, will lead to points that are arbitrarily far apart (subject to the confines of the attractor), and after any of various other numbers of iterations will lead to points that are arbitrarily close together. Thus a dynamic system with a chaotic attractor is locally unstable yet globally stable: once some sequences have entered the attractor, nearby points diverge from one another but never depart from the attractor.</p>
<p>The term <b>strange attractor</b> was coined by <a href="/wiki/David_Ruelle" title="David Ruelle">David Ruelle</a> and <a href="/wiki/Floris_Takens" title="Floris Takens">Floris Takens</a> to describe the attractor resulting from a series of <a href="/wiki/Bifurcation_theory" title="Bifurcation theory">bifurcations</a> of a system describing fluid flow. Strange attractors are often <a class="mw-redirect" href="/wiki/Differentiable" title="Differentiable">differentiable</a> in a few directions, but some are <a class="mw-redirect" href="/wiki/Homeomorphic" title="Homeomorphic">like</a> a <a class="mw-redirect" href="/wiki/Cantor_dust" title="Cantor dust">Cantor dust</a>, and therefore not differentiable. Strange attractors may also be found in presence of noise, where they may be shown to support invariant random probability measures of Sinai–Ruelle–Bowen type.</p>
<p>Examples of strange attractors include the <a class="mw-redirect" href="/wiki/Double_scroll_attractor" title="Double scroll attractor">double-scroll attractor</a>, <a href="/wiki/H%C3%A9non_map" title="Hénon map">Hénon attractor</a>, <a href="/wiki/R%C3%B6ssler_attractor" title="Rössler attractor">Rössler attractor</a>, Tamari attractor, and the <a class="mw-redirect" href="/wiki/Lorenz_attractor" title="Lorenz attractor">Lorenz attractor</a>.</p>

<h2><span class="mw-headline" id="Basins_of_attraction">Basins of attraction</span></h2>
<p>An attractor's <b>basin of attraction</b> is the region of the <a href="/wiki/Phase_space" title="Phase space">phase space</a>, over which iterations are defined, such that any point (any <a href="/wiki/Initial_condition" title="Initial condition">initial condition</a>) in that region will eventually be iterated into the attractor. For a <a class="mw-redirect" href="/wiki/Stability_(mathematics)" title="Stability (mathematics)">stable</a> <a href="/wiki/Linear_system" title="Linear system">linear system</a>, every point in the phase space is in the basin of attraction. However, in <a href="/wiki/Nonlinear_system" title="Nonlinear system">nonlinear systems</a>, some points may map directly or <a class="mw-redirect" href="/wiki/Asymptotic_behavior" title="Asymptotic behavior">asymptotically</a> to infinity, while other points may lie in a different basin of attraction and map asymptotically into a different attractor; other initial conditions may be in or map directly into a non-attracting point or cycle.</p>

<h3><span class="mw-headline" id="Nonlinear_equation_or_system">Nonlinear equation or system</span></h3>
<p>Equations or systems that are <a href="/wiki/Nonlinear_system" title="Nonlinear system">nonlinear</a> can give rise to a richer variety of behavior than can linear systems. One example is <a href="/wiki/Newton%27s_method" title="Newton's method">Newton's method</a> of iterating to a root of a nonlinear expression. If the expression has more than one <a href="/wiki/Real_number" title="Real number">real</a> root, some starting points for the iterative algorithm will lead to one of the roots asymptotically, and other starting points will lead to another. The basins of attraction for the expression's roots are generally not simple—it is not simply that the points nearest one root all map there, giving a basin of attraction consisting of nearby points. <h2><span class="mw-headline" id="Partial_differential_equations">Partial differential equations</span></h2>
<p><a href="/wiki/Parabolic_partial_differential_equation" title="Parabolic partial differential equation">Parabolic partial differential equations</a> may have finite-dimensional attractors. The diffusive part of the equation damps higher frequencies and in some cases leads to a global attractor. The <i>Ginzburg–Landau</i>, the <i>Kuramoto–Sivashinsky</i>, and the two-dimensional, forced <a class="mw-redirect" href="/wiki/Navier%E2%80%93Stokes_equation" title="Navier–Stokes equation">Navier–Stokes equations</a> are all known to have global attractors of finite dimension.</p>
<p>For the three-dimensional, incompressible Navier–Stokes equation with periodic <a class="mw-redirect" href="/wiki/Boundary_condition" title="Boundary condition">boundary conditions</a>, if it has a global attractor, then this attractor will be of finite dimensions.</p>
<h2><span class="mw-headline" id="Numerical_localization_.28visualization.29_of_attractors:_self-excited_and_hidden_attractors">Numerical localization (visualization) of attractors: self-excited and hidden attractors</span></h2>

<p>From a computational point of view, attractors can be naturally regarded as <i>self-excited attractors</i> or <i><a class="mw-redirect" href="/wiki/Hidden_attractor" title="Hidden attractor">hidden attractors</a></i>. Self-excited attractors can be localized numerically by standard computational procedures, in which after a transient sequence, a trajectory starting from a point on an unstable manifold in a small neighborhood of an unstable equilibrium reaches an attractor (like classical attractors in the <a href="/wiki/Van_der_Pol" title="Van der Pol">Van der Pol</a>, <a href="/wiki/Belousov%E2%80%93Zhabotinsky_reaction" title="Belousov–Zhabotinsky reaction">Belousov–Zhabotinsky</a>, <a class="mw-redirect" href="/wiki/Lorenz_attractor" title="Lorenz attractor">Lorenz</a>, and many other dynamical systems). In contrast, the basin of attraction of a <a class="mw-redirect" href="/wiki/Hidden_attractor" title="Hidden attractor">hidden attractor</a> does not contain neighborhoods of equilibria, so the <a class="mw-redirect" href="/wiki/Hidden_attractor" title="Hidden attractor">hidden attractor</a> cannot be localized by standard computational procedures.</p>
