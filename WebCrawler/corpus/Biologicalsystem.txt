<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><p>A <b>biological system</b> is a complex <a href="/wiki/Biological_network" title="Biological network">network</a> of biologically relevant <a class="mw-redirect" href="/wiki/Entities" title="Entities">entities</a>. As biological organization spans several scales, examples of biological systems are <a class="mw-redirect" href="/wiki/Populations" title="Populations">populations</a> of <a class="mw-redirect" href="/wiki/Organisms" title="Organisms">organisms</a>, or on the <a href="/wiki/Organ_(anatomy)" title="Organ (anatomy)">organ</a>- and <a href="/wiki/Tissue_(biology)" title="Tissue (biology)">tissue</a> scale in <a class="mw-redirect" href="/wiki/Mammals" title="Mammals">mammals</a> and other animals, the <a href="/wiki/Circulatory_system" title="Circulatory system">circulatory system</a>, the <a href="/wiki/Respiratory_system" title="Respiratory system">respiratory system</a>, the <a href="/wiki/Nervous_system" title="Nervous system">nervous system</a>, etc.</p>
<p>On the <a href="/wiki/Micrometre" title="Micrometre">micro</a> to the <a href="/wiki/Nanometre" title="Nanometre">nano</a>scopic scale, examples of biological systems are <a class="mw-redirect" href="/wiki/Cells" title="Cells">cells</a>, <a class="mw-redirect" href="/wiki/Organelles" title="Organelles">organelles</a>, macromolecular complexes and regulatory pathways.</p>
<p>A biological system is not to be confused with a <a href="/wiki/Living_systems" title="Living systems">living system</a>, which is commonly referred to as <a href="/wiki/Life" title="Life">life</a>. For further information see e.g. <a class="mw-redirect" href="/wiki/Definition_of_life" title="Definition of life">definition of life</a> or <a href="/wiki/Synthetic_biology" title="Synthetic biology">synthetic biology</a>.and you can also visit totora, gyton and several many books for details</p>
<h2><span class="mw-headline" id="Organ_and_tissue_systems">Organ and tissue systems</span></h2>

<p>These specific systems are widely studied in <a class="mw-redirect" href="/wiki/Human_anatomy" title="Human anatomy">Human anatomy</a>. <i>"Human" systems</i> are also present in many other animals.</p>
<ul>
<li><a href="/wiki/Circulatory_system" title="Circulatory system">Circulatory system</a>: pumping and channeling <a href="/wiki/Blood" title="Blood">blood</a> to and from the body and <a class="mw-redirect" href="/wiki/Lungs" title="Lungs">lungs</a> with <a href="/wiki/Heart" title="Heart">heart</a>, blood and <a href="/wiki/Blood_vessel" title="Blood vessel">blood vessels</a>.</li>
<li><a href="/wiki/Integumentary_system" title="Integumentary system">Integumentary system</a>: <a href="/wiki/Skin" title="Skin">skin</a>, <a href="/wiki/Hair" title="Hair">hair</a>, <a href="/wiki/Fat" title="Fat">fat</a>, and <a href="/wiki/Nail_(anatomy)" title="Nail (anatomy)">nails</a>.</li>
<li><a class="mw-redirect" href="/wiki/Skeletal_system" title="Skeletal system">Skeletal system</a>: structural support and protection with <a href="/wiki/Bone" title="Bone">bones</a>, <a href="/wiki/Cartilage" title="Cartilage">cartilage</a>, <a href="/wiki/Ligament" title="Ligament">ligaments</a> and <a href="/wiki/Tendon" title="Tendon">tendons</a>.</li>
<li><a href="/wiki/Reproductive_system" title="Reproductive system">Reproductive system</a>: the <a class="mw-redirect" href="/wiki/Sex_organs" title="Sex organs">sex organs</a>, such as <a href="/wiki/Ovary" title="Ovary">ovaries</a>, <a href="/wiki/Fallopian_tube" title="Fallopian tube">fallopian tubes</a>, <a href="/wiki/Uterus" title="Uterus">uterus</a>, <a href="/wiki/Vagina" title="Vagina">vagina</a>, <a href="/wiki/Mammary_gland" title="Mammary gland">mammary glands</a>, <a class="mw-redirect" href="/wiki/Testes" title="Testes">testes</a>, <a href="/wiki/Vas_deferens" title="Vas deferens">vas deferens</a>, <a href="/wiki/Seminal_vesicle" title="Seminal vesicle">seminal vesicles</a> and <a href="/wiki/Prostate" title="Prostate">prostate</a></li>
<li><a class="mw-redirect" href="/wiki/Digestive_system" title="Digestive system">Digestive system</a>: <a href="/wiki/Digestion" title="Digestion">digestion</a> and processing food with <a href="/wiki/Salivary_gland" title="Salivary gland">salivary glands</a>, <a href="/wiki/Esophagus" title="Esophagus">esophagus</a>, <a href="/wiki/Stomach" title="Stomach">stomach</a>, <a href="/wiki/Liver" title="Liver">liver</a>, <a href="/wiki/Gallbladder" title="Gallbladder">gallbladder</a>, <a href="/wiki/Pancreas" title="Pancreas">pancreas</a>, <a class="mw-redirect" href="/wiki/Intestine" title="Intestine">intestines</a>, <a href="/wiki/Rectum" title="Rectum">rectum</a> and <a href="/wiki/Anus" title="Anus">anus</a>.</li>
<li><a href="/wiki/Urinary_system" title="Urinary system">Urinary system</a>: <a href="/wiki/Kidney" title="Kidney">kidneys</a>, <a href="/wiki/Ureter" title="Ureter">ureters</a>, <a href="/wiki/Urinary_bladder" title="Urinary bladder">bladder</a> and <a href="/wiki/Urethra" title="Urethra">urethra</a> involved in fluid balance, <a href="/wiki/Electrolyte" title="Electrolyte">electrolyte</a> balance and excretion of <a href="/wiki/Urine" title="Urine">urine</a>.</li>
<li><a href="/wiki/Respiratory_system" title="Respiratory system">Respiratory system</a>: the organs used for <a href="/wiki/Breathing" title="Breathing">breathing</a>, the <a class="mw-redirect" href="/wiki/Human_pharynx" title="Human pharynx">pharynx</a>, <a href="/wiki/Larynx" title="Larynx">larynx</a>, <a class="mw-redirect" href="/wiki/Bronchi" title="Bronchi">bronchi</a>, <a href="/wiki/Lung" title="Lung">lungs</a> and <a href="/wiki/Thoracic_diaphragm" title="Thoracic diaphragm">diaphragm</a>.</li>
<li><a href="/wiki/Endocrine_system" title="Endocrine system">Endocrine system</a>: communication within the body using <a href="/wiki/Hormone" title="Hormone">hormones</a> made by <a href="/wiki/Endocrine_gland" title="Endocrine gland">endocrine glands</a> such as the <a href="/wiki/Hypothalamus" title="Hypothalamus">hypothalamus</a>, <a class="mw-redirect" href="/wiki/Pituitary" title="Pituitary">pituitary</a> gland, <a class="mw-redirect" href="/wiki/Pineal_body" title="Pineal body">pineal body</a> or pineal gland, <a href="/wiki/Thyroid" title="Thyroid">thyroid</a>, <a class="mw-redirect" href="/wiki/Parathyroid" title="Parathyroid">parathyroid</a> and <a class="mw-redirect" href="/wiki/Adrenal" title="Adrenal">adrenals</a>, i.e., adrenal glands.</li>
<li><a href="/wiki/Immune_system" title="Immune system">Immune system</a>: protects the organism from foreign bodies
<ul>
<li><a href="/wiki/Lymphatic_system" title="Lymphatic system">Lymphatic system</a>: structures involved in the transfer of <a href="/wiki/Lymph" title="Lymph">lymph</a> between <a href="/wiki/Tissue_(biology)" title="Tissue (biology)">tissues</a> and the <a class="mw-redirect" href="/wiki/Blood_stream" title="Blood stream">blood stream</a>; includes the lymph and the <a href="/wiki/Lymph_node" title="Lymph node">nodes</a> and <a class="mw-redirect" href="/wiki/Lymph_vessel" title="Lymph vessel">vessels</a>. The lymphatic system includes functions including immune responses and development of antibodies.</li>
</ul>
</li>
<li><a href="/wiki/Muscular_system" title="Muscular system">Muscular system</a>: allows for manipulation of the environment, provides locomotion, maintains posture, and produces heat. Includes <a class="mw-redirect" href="/wiki/Skeletal_muscles" title="Skeletal muscles">skeletal muscles</a>, <a class="mw-redirect" href="/wiki/Smooth_muscles" title="Smooth muscles">smooth muscles</a> and <a href="/wiki/Cardiac_muscle" title="Cardiac muscle">cardiac muscle</a>.</li>
<li><a href="/wiki/Nervous_system" title="Nervous system">Nervous system</a>: collecting, transferring and processing information with <a href="/wiki/Brain" title="Brain">brain</a>, <a href="/wiki/Spinal_cord" title="Spinal cord">spinal cord</a> and <a href="/wiki/Peripheral_nervous_system" title="Peripheral nervous system">peripheral nervous system</a>.</li>
</ul>
